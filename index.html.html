如何取得 Google Drive 檔案 ID

  1. 在 Google Drive 找到記帳備份 .json 檔案
  2. 右鍵 → 「取得連結」
  3. 連結格式：…/file/d/這段是ID/view
  4. 把中間那段 ID 貼到輸入框

第一次使用：在 Google Drive 建立空的 family-budget-2026.json，記下 ID 即可。
